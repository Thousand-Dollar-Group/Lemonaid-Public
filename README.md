Lemonaid is a full-stack solution designed to streamline the supply chain quoting process. This project transforms a workflow that traditionally takes **weeks** into a matter of **30 minutes**, providing a competitive edge through speed and efficiency.

- **Impact:** Reduced supply chain quoting time from weeks to **30 minutes**.

- **Partnership:** Developed in partnership with a **fastener distributors** for data provision and business value validation (partnership is now concluded).
- **Development Team:** Built by an **independent volunteer development team** as a portfolio/project initiative.

---

## üõ†Ô∏è Technical Stack & Architecture

Lemonaid is built on a robust, modern, and scalable architecture leveraging best-in-class open-source tools and AWS services.

### Backend (FastAPI / Python)

- **Framework:** **FastAPI** (Python) for high performance and rapid development.
- **Database:** **PostgreSQL** for reliable, structured data management.
- **Core Features:**
  - **Intelligent Chatbot Feature**
  - **Query and Drawing Parsing Pipeline** (Interpreting complex requirements)
  - Secure **User Authentication**
  - Comprehensive **Conversation History** logging

### Frontend (React / TypeScript)

- **Framework:** **React** powered by **TypeScript** for scalable and maintainable user interfaces.
- **Core Features:**
  - Highly **Interactive Chatbot Interface**
  - Seamless **User Authentication** flow
  - Detailed **Conversation History** view
  - **Email Preview** generation
  - **Attachment Preview** functionality

### Cloud Infrastructure (AWS)

The system is deployed on a highly available, serverless architecture using Amazon Web Services (AWS):

- **API Gateway** & **Lambda:** Serverless computation for API endpoints.
- **AWS Cognito:** Managed service for user authentication and authorization.
- **AWS RDS (PostgreSQL):** Managed relational database service.
- **S3:** Scalable object storage for attachments and data.
- **AWS Amplify:** Frontend hosting and deployment.
